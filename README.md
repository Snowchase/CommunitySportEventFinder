# Project Name: Community Sport Event Finder Project
## Authors: Robert Chase Edie, Tharun Kumar Malla Dinakaran, Tanner Lyman, Jack Vermaak
### Group: 3
### Created: 2/22/26
### Description: 
            This project will create an Java application using maven that allows users
            to connect with other local athletes/community members to play
            casual sports together like basketball, tennis, golf, and even bowling.

            We will use Swing to incorporate our UI elements
            
	          This project will use SQLite to allow authenticating 
            users and storing user data. This can later be 
            used for the friend system and/or event leaderboards/stats

## Features to be implemented Primarily:
###    -Login system for users to safely access their information
###    -User should have a page allowing them to change aspects of the created account
        -Information like name, age, gender, location(Coarse)
        -Allow to add sport preferences and positions
        -Allow users to add estimated skill levels
###    -Sport event creation, browser, and summaries given to authorized user
###    -Sport event creators should be able to easily create and manage their sessions
        -Features like sport type, date, start time, location, max players, public/private
        -Certain sports will have positions and/or fees
        -Min/Max skill level caps can be implemented for certain events
        - high level parameters in sports, kick feature, event started button
        -Events should reflect on created community browser after create button is pressed by even creator
###    -Sport Event browser should clearly show a list of nearby upcoming sporting events for preferred sports
        -Sessions can include sport type, date, start time, location, max players, and spots remaining
        -Certain sports will need positions, Skill level caps, and/or fees
        -Users should be able to filter by various categories such as sport, skill level, distance, date, and cost
            -Interactive map created later should reflect filtered sports browser
